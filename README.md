# Painel Econômico Focus

Projeto desenvolvido para análise de indicadores econômicos utilizando dados públicos do Relatório Focus do Banco Central.

O sistema realiza automaticamente:

- Acesso ao site do Banco Central via Selenium
- Download automatizado do relatório Focus em PDF
- Extração e tratamento dos dados com Python
- Geração de arquivo CSV estruturado
- Visualização dos indicadores em dashboard Power BI

## Indicadores analisados

- Selic
- IPCA
- PIB Total
- Câmbio
- IGP-M

## Tecnologias utilizadas

- Python
- Selenium
- PDFPlumber
- Pandas
- Power BI

## Funcionalidades do Dashboard

- Comparativo entre períodos
- Filtro por ano
- Indicadores de variação econômica
- Atualização automatizada dos dados
